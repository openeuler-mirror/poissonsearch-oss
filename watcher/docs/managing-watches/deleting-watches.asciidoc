[[deleting-watches]]
=== Deleting Watches

You use the Watcher <<api-rest-delete-watch, `delete`>> API to permanently remove a watch. 

For example:

[source,js]
--------------------------------------------------
DELETE _watcher/watch/my_watch
--------------------------------------------------
// AUTOSENSE

A successful response looks like this:

[source,js]
--------------------------------------------------
{
   "_id": "my-watch",
   "_version": 8,
   "found": true
}
--------------------------------------------------

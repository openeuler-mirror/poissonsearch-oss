[[watcher-api-restart]]
=== Restart API

The `restart` API stops, then starts the {watcher} service, as in the
following example:

[source,js]
--------------------------------------------------
POST _xpack/watcher/_restart
--------------------------------------------------
// CONSOLE

{watcher} returns the following response if the request is successful:

[source,js]
--------------------------------------------------
{
   "acknowledged": true
}
--------------------------------------------------
// TESTRESPONSE
